# react-100-knock

## 目的

Reactに慣れる

## 内容

1. Hello World App
- シンプルなReactアプリを構築し、"Hello, World!"を表示する。

2. カウンターアプリ
- 数を増減させるカウンターアプリ。

3. タイマーアプリ
- カウントダウンタイマーを設定できるアプリ。

4. To-Do リスト
- タスクを追加、削除できる基本的なTo-Doリストアプリ。

5. ノートアプリ
- ユーザーがノートを追加、編集、削除できる。

6. ブログアプリ
- ユーザーがブログポストを作成、表示できる。

7. クイズアプリ
- クイズの質問と回答を表示し、スコアを追跡するアプリ。

8. カレンダーアプリ
- イベントの予定を表示するカレンダーアプリ。

9. グラフ表示アプリ
- データをグラフで視覚化するアプリ。

10. チャットアプリ
- リアルタイムのチャット機能を持つアプリ。

11. 画像ギャラリーアプリ
- 画像をアップロードしてギャラリーで表示するアプリ。

12. ニュースアプリ
- 最新のニュース記事を表示するアプリ。

13. 音楽プレイヤーアプリ
- 音楽を再生、一時停止、スキップできるアプリ。

14. ショッピングカートアプリ
- 製品をカートに追加し、注文を確定できるアプリ。

15. ソーシャルメディアアプリ
- ユーザーがプロフィールを作成し、投稿できるソーシャルメディアアプリ。

16. オンライン予約システム
- ユーザーが予約を作成し、管理できるアプリ。

17. ワードクラウドジェネレーター
- テキストデータからワードクラウドを生成するアプリ。

18. ブラウザゲーム
- シンプルなウェブゲームを作成する。

19. 天気予報アプリ
- 現在の天気情報と予報を表示するアプリ。

20. 位置情報ベースのアプリ
- ユーザーの現在位置に関連する情報を提供するアプリ。
